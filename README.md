Hybrid renewable energy-based dynamic EV charging system using solar PV and ducted wind energy with real-time energy management.
# Hybrid Photovoltaic and Ducted Energy Integration for Dynamic EV Charging

## 📌 Overview
This project presents a hybrid renewable energy-based dynamic charging system for electric vehicles (EVs). It integrates solar photovoltaic (PV) and ducted wind energy to enable continuous charging during vehicle motion, reducing dependency on stationary charging stations.

## 🚀 Objective
- To enable on-the-go EV charging using renewable energy
- To improve charging efficiency and reliability
- To ensure battery safety using real-time monitoring

## ⚙️ Technologies Used
- Solar Photovoltaic System
- Ducted Wind Energy System
- Arduino Nano (Microcontroller)
- Sensors (Voltage, Temperature)
- Power Electronics (DC-DC Converter)
- Embedded C / C++

## 🔋 Key Features
- Hybrid energy generation (solar + wind)
- Dynamic Energy Management System (DEMS)
- Real-time monitoring of battery parameters
- Automatic charging control based on State of Charge (SoC)
- Improved charging continuity and efficiency

## 📊 Results
- Output voltage range: 12V – 18V
- Improved charging continuity by 30–35%
- Reduced dependency on fixed charging infrastructure

## 🌱 Applications
- Electric Vehicles (EVs)
- Sustainable transportation
- Smart energy systems
